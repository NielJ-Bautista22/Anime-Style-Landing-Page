Anime Landing Page – Code Geass

Welcome to Anime Landing Page – Code Geass, a web project inspired by the iconic anime Code Geass. This project is a personal practice in HTML, CSS, and JavaScript, focusing on creating dynamic and interactive web elements.

-Project Overview

This landing page is designed with a modern anime aesthetic, highlighting key elements of Code Geass. The page is structured with the following sections:

Hero Section: Engaging top section with dynamic images of characters. Features two different hero images that appear randomly on page load.

Summary: Brief overview of the anime storyline and main themes.

Scenes: Visual showcase of important scenes and moments.

Characters: Interactive section displaying main characters, their traits, and affiliations.

-Features

Dynamic Hero Sections: Each time the page is loaded, a different hero image appears, keeping the experience fresh.

Scrolling Animations: Smooth animations while scrolling through sections for a modern web experience.

Never-ending Carousel: Automatically scrolling carousel to showcase characters or scenes, looping infinitely.

-Purpose

This project was built as part of my front-end practice. It helped me strengthen skills in:

HTML structure and semantic markup

CSS layout, animations, and responsive design

JavaScript DOM manipulation and interactive elements

-Technologies Used

HTML5 – For semantic content and page structure

CSS3 – For styling, animations, and responsive layouts

JavaScript – For interactive elements, random hero display, and carousel functionality
